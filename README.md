# What's here

Right now, BGP related info for Cisco.

* Cisco - BGP Update stats
* Cisco - BGP Prefix counts

# How to install

* Copy `cisco_bgp.xml` to `/usr/share/cacti/resource/snmp_queries`.
* Import `cacti_*.xml` via web-interface.
